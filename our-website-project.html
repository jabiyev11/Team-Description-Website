Our Website Project



Plan Overview

We as a Team Matrix came together as soon as the Homework 3 published and started to work on this project. Firstly, we assigned the roles and divided the website to 4 sections as Main page, Our team, Hour of Code and Website project. We used different html, css and js files and connected them. We searched for the different styles in Figma website to see different structures. After that we visualized the structure of the pages using Ipod and started to work on the technical part of this. We used different extentions in VS code such as live server, code snap and live preview to make the process easy for us. We arranged meetings almost every day to be on track and finish the assignment on time.


    Homepage

  1. Homepage (HTML, CSS, JavaScript):

    - HTML represents webpage structure, including navigation, main title section, and canvas element.

    - External CSS and JavaScript files are linked.

    - Navigation bar contains links to different pages.

    - Main title section includes heading, paragraph, and quote.

    - CSS file styles body, hero section, main title, navigation menu, and more.

    - JavaScript (index.js) handles navigation menu on scroll, theme switching, and page transitions.


    2. Our Team (HTML, CSS, JavaScript):

  2. - HTML follows standard structure with meta tags, links, navigation, team members' info, and footer.

    - CSS styles layout, fonts, headings, paragraphs, images, lists, and containers.

    - Team member info is structured with images, text, and contact details.

    - Footer contains links to ADA University website and GitHub repository.

    - External JavaScript file (index.js) is linked.

    - Features include a "scroll up" button and a theme icon.


    3. Hour of Code (HTML, CSS, JavaScript):

  3. - HTML structure with standard tags, meta tags, title, and links.

    - Navigation bar with logo and links, content section with heading, slider, and paragraph.

    - Footer with links and an image.

    - CSS styles body, slider, transitions, paragraph container, centered image, footer, and links.

    - JavaScript at the end for automatic slider transitions using setInterval.


    4. Global CSS:

  4. - Variables: Defines colors and styles with custom properties.

    - Global Styles: Applies common font and basic styles to all elements with smooth transitions.

    - Link Styles: Defines link appearance and changes color on hover.

    - Body Styles: Sets background color, font, and top margin.

    - Container Styles: Limits content width and centers it.

    - Navbar Styles: Styles for a fixed-position navbar with logo, menu, and theme icon.

    - Navigation Underline: Adds an underline to navigation links that grows on hover.

    - Scroll-Behavior: Enables smooth scrolling.

    - Footer Styles: Adds a fixed-height footer with a box shadow.

    - Button Styles: Styles for a "Go to Top" button.

    - Image Filters: Applies filters that change on hover.

    - Page Transition: Includes a fade-out effect for transitioning between pages.
